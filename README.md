# ring0

Kernel incident training RPG. Learn Linux kernel internals through maze puzzles and CTF challenges.

## What is this?

ring0 is a game where you navigate maze levels, interact with kernel components, and solve real CVE-based challenges. It covers topics like memory management, process scheduling, filesystem internals, and common kernel vulnerabilities.

## Features

- 10 training levels that teach kernel concepts step by step
- 14 CTF levels based on real CVE-2025 kernel vulnerabilities
- 3 enemy types with different AI behaviors
- Terminal-based interaction and flag submission
- Kernel boot themed UI

## Download

Grab the latest EXE from [Releases](https://github.com/0xCD4/kernel-rpg/releases). No installation needed, just run it.

## Controls

- **WASD / Arrow keys** - Move
- **E** - Interact with objects
- **H** - Toggle help overlay
- **T** - Open terminal (CTF mode)
